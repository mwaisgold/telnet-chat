telnet-chat
===========

How to run it:

1) Install gvm:
  curl -s get.gvmtool.net | bash

2) Install groovy:
  gvm install groovy 2.1.6
  (Mark as default version)

3) Clone this project:
  git clone https://github.com/mwaisgold/telnet-chat.git

4) Go into src folder:
  cd src/

5) Run the server:
  groovy base.groovy
